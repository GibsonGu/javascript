A)Instruction - top

  1. Drop folder "drag2share" to your directory.
  2. In your document header section, include Jquery, Jquery UI and drag2share javascript files and drag2share.css. 
    <head>
	......
	<script type="text/javascript" src="drag2share/js/jquery-1.3.2.min.js"></script>
	<script type="text/javascript" src="drag2share/js/jquery-ui-1.7.2.custom.min.js"></script>
	<script type="text/javascript" src="drag2share/js/drag2share.js"></script>
	<link rel="stylesheet" type="text/css" href="drag2share/dragToShare.css">
	......
</head>
  3. Call drag2share to initialize the action 
    <script type="text/javascript">
 $(function() {
	    $('.images-class').drag2share();
});
</script>

B) Options - top

Key Default Description
icons "icons" Path to icons' directory, you can edit this property if you would like to move DragToShare's icons to your application's image folder.
revertDrag false When it is set to true, helper will revert back to his original place.
targets "linkedin,technorati,delicious,yahoo,google,facebook,twitter,stumble" Set this property to where you want your users to share this page. You can also edit this property to arrange the sequence of the shared icons. Currently, there are eight available options, we are working on more icons.
